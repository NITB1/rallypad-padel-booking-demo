# RallyPad

Mobile-first padel court booking demo built for pitch video and ad b-roll.

## Brand

- Name: RallyPad
- Positioning: "Run your club. Fill every court."
- Logo: padel racket, calendar square, and ball
- Palette: deep court green `#0c3b2e`, turf green `#1f8a5b`, tennis-ball lime `#c7f24b`, warm coral `#ff7056`, white surfaces

## Demo Sections

- Court booking schedule with court filters and check-in states
- Follow-up clients with call/message actions
- Smart reminders for check-ins, court starts, and payments
- Loyalty e-punch card with 10-pass progress
- Club pulse panel with revenue, occupancy, and top time slot
- Growth tools for campaigns, reminders, punch cards, and rewards

## Local Development

```bash
npm install
npm run dev
```

## Production

```bash
npm run build
npm start
```

The `start` script serves the Vite `dist` folder and reads Railway's `PORT` environment variable.
